Lets learn about HTML

HTML HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of web pages.

Lets learn about CSS

CSS Cascading Style Sheets is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, and fonts of web pages.

Lets Learn about JavaScript

JavaScript JavaScript is a programming language that is commonly used to create interactive web pages. It is a client-side scripting language that runs in the web browser.
